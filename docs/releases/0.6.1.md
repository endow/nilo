## Release Notes (English)

Nilo 0.6.1 is a patch release focused on release verification and roadmap UX. It clarifies the full-check requirement before release publication, reusable checks for metadata-only changes, and staged changed checks during release preparation.

### Changes

- Required a full check before release publish.
- Allowed release full checks to be reused for metadata-only changes.
- Staged release prepare around a changed check, with full verification deferred to publish when appropriate.
- Added slowest-shard reporting for test runtime visibility.
- Refined roadmap, overdrive, and status-surface output and progression scope.
- Stabilized project summary guidance coverage.

### Evidence

- Change range: `v0.6.0..v0.6.1`
