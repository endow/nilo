# v0.6.3

## リリースノート（日本語）

Nilo 0.6.3 は、`nilo work` を通常作業の既定入口として追加する patch release です。作業開始、recipe 選択、検証記録、AI 完了報告をひとつの入口から扱いやすくしました。

### 変更点

- `nilo work` default entrypoint を追加しました。
- 通常作業で recipe を明示しない場合でも、Nilo が依頼内容から作業セッションを開始しやすくしました。

### 根拠

- 変更範囲: `v0.6.2..v0.6.3`
- 対象コミット: `b2d082b`, `904c064`
- 公開は人間承認後に tag / push / GitHub release を実行。

## Release Notes (English)

Nilo 0.6.3 is a patch release that adds `nilo work` as the default entrypoint for normal work. It makes task startup, recipe selection, verification recording, and AI completion reporting easier to reach from one command.

### Changes

- Added the `nilo work` default entrypoint.
- Made it easier to start a Nilo work session from a normal request without choosing a recipe explicitly.

### Evidence

- Change range: `v0.6.2..v0.6.3`
- Commits: `b2d082b`, `904c064`
- Publishing remains gated by explicit human approval for tag, push, and GitHub release creation.
