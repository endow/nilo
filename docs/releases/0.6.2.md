# v0.6.2

## リリースノート（日本語）

Nilo 0.6.2 は、release recipe の責務分離と Claude CLI レビュー導線を追加する patch release です。release recipe が public operation に進む前の責務を明確にし、通常のレビュー入口から Claude CLI を使えるようにしました。

### 変更点

- release recipe の prepare / publish の責務分離を実装しました。
- Claude CLI を直接レビュー導線として使う path を追加しました。
- release full check のレビュー導線テスト失敗を修正しました。

### 根拠

- 変更範囲: `v0.6.1..v0.6.2`
- 対象コミット: `0750a90`, `634b7d6`, `e677e58`, `0b63daa`
- 公開は人間承認後に tag / push / GitHub release を実行。

## Release Notes (English)

Nilo 0.6.2 is a patch release that separates release recipe responsibilities and adds a direct Claude CLI review path. It clarifies what release preparation owns before public operations and makes Claude CLI available through the normal review route.

### Changes

- Implemented the prepare / publish responsibility split for the release recipe.
- Added a direct Claude CLI review path.
- Fixed the release full-check review-flow test failure.

### Evidence

- Change range: `v0.6.1..v0.6.2`
- Commits: `0750a90`, `634b7d6`, `e677e58`, `0b63daa`
- Publishing remains gated by explicit human approval for tag, push, and GitHub release creation.
