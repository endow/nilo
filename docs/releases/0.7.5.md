# v0.7.5

## リリースノート（日本語）

### 変更点

- 読み取り専用依頼を不要に Task 化せず、通常作業の受付・検証・完了フローを簡素化した。
- 未完了 Task を安全に終了する `nilo cancel` コマンドを追加した。
- cancelled Task が queue と status の active 候補へ再表示されないようにした。

- Prevented cancelled tasks from reappearing as active candidates in queue and status output.
