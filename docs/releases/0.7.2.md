# v0.7.2

## リリースノート（日本語）

Nilo 0.7.2 は、リリース証跡の鮮度判定と環境差分に起因するテストの安定性を改善するパッチリリースです。

### 変更点

- 検証済みの release commit を現在の証跡として扱えるようにし、release metadata だけの変更で full verification を再利用できる判定を強化した。
- macOS 環境で path assertion が不安定にならないよう、release workflow / upgrade 周辺テストのパス比較を安定化した。
- `nilo view` が古い schema の stale データに遭遇した場合の表示エラーを修正した。

## Release Notes (English)

Nilo 0.7.2 is a patch release that improves release evidence freshness handling and test stability across environments.

### Changes

- Treat verified release commits as current evidence and strengthen the reuse path for full verification when only release metadata changed.
- Stabilize release workflow and upgrade path assertions on macOS.
- Fix a `nilo view` display error when stale data uses an older schema.
