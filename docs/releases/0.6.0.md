# v0.6.0

## リリースノート（日本語）

Nilo 0.6.0 は、人間ゲート付きタスクの扱いと完了診断を強める patch release です。ロードマップ表示の待ち時間を減らし、AI が人間承認待ちの作業を完了扱いにしないようにしました。

### 変更点

- 人間の受け入れ判断が必要なタスクを、AI が自動完了として閉じないようにしました。
- 完了診断と roadmap summary の性能を改善しました。
- release workflow のテスト期待値を、現在の挙動に合わせて修正しました。

### 根拠

- 変更範囲: `v0.5.8..v0.6.0`
- 対象コミット: `32bcbed`, `c8206e4`, `2432cc8`, `646f9c9`
- 公開は人間承認後に tag / push / GitHub release を実行。

## Release Notes (English)

Nilo 0.6.0 is a patch release that strengthens human-gated task handling and completion diagnostics. It reduces roadmap-summary latency and prevents AI agents from closing work that still requires human acceptance.

### Changes

- Prevented AI auto-completion of tasks that require human acceptance.
- Improved completion diagnostics and roadmap summary performance.
- Updated release workflow test expectations to match current behavior.

### Evidence

- Change range: `v0.5.8..v0.6.0`
- Commits: `32bcbed`, `c8206e4`, `2432cc8`, `646f9c9`
- Publishing remains gated by explicit human approval for tag, push, and GitHub release creation.
