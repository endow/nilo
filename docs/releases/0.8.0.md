## Release Notes (English)

### Changes

- Refactored workflow responsibilities and moved state decisions into projection-driven use cases.
- Separated completion projections from historical indeterminate state, and made cancelled tasks terminal for roadmap evidence.
- Extracted review dispatch behind an external adapter boundary to clarify review execution responsibilities.
- Documented the schema inventory and concept model, and removed the generated `ROADMAP.md` surface.
